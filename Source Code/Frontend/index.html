A Student Intelligent Tutoring System

This System builds an expert-agnostic student Intelligent Tutoring System using self-supervised Deep Learning. This System validate the tutoring information of the student and to predict student performance without manual annotations. And achieved about 98% accuracy by using the hybrid GRU-CNN and CNN models, this system effectively capturing both short-term and long-term learning patterns of tutoring information. The system delivers real-time, personalized feedback, enabling scalable and cost-effective AI-driven education across diverse subjects.

This research contributes to the development of Student intelligent tutoring systems through innovative AI approaches that minimize reliance on domain experts.
